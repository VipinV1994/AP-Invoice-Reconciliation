## Business Value

These tools are designed to:  

- Reduce duplicate payments and overpayments  
- Improve audit readiness and compliance  
- Strengthen financial controls and evidence trails  
- Automate aging and risk analysis  
- Provide actionable insights via dashboards for decision-making  

---

## Project Overview

| Project | Purpose | Key Features |
|---------|---------|--------------|
| **Corporate Card Reconciliation** | Monitor card spend and policy compliance | Over-limit alerts, missing receipt detection, duplicate transaction flags, pivot dashboards |
| **Duplicate Payment Detection** | Detect duplicate invoices and reduce financial risk | Conditional logic with formulas, summary reports, visual alerts |
| **Invoice Reconciliation & Aging** | Match invoices to payments and track overdue items | Aging analysis, discrepancy flags, pivot dashboards, department-level reporting |
| **Vendor Spend & Risk Tracker** | Monitor vendor payments and trends | Vendor scorecards, overdue invoices, spend analysis, dashboards |
