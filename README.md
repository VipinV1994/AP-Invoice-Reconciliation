# AP Invoice Reconciliation Template

This project demonstrates an accounts payable invoice reconciliation workflow using Excel.

## Purpose
To reconcile vendor invoices against payments and identify:
- Unpaid invoices
- Partial payments
- Overpayments
- Duplicate payments

## Features
- Invoice and payment registers
- Automated reconciliation using Excel formulas
- Difference calculation and status flags
- Designed for corporate AP environments

## Tools Used
- Microsoft Excel
- SUMIFS, IF formulas
- Pivot Tables (optional)

## Use Case
This template reflects real-world AP processes such as:
- Monthly vendor reconciliation
- Audit support
- Payment accuracy checks
